🎯 PROBLEMA RESOLVIDO:
   ❌ ANTES: Mostrava o projeto todo
   ✅ AGORA: Mostra apenas o desenho da barra específica

📋 COMO FUNCIONA:

1. BUSCA INTELIGENTE:
   • Procura textos com nome EXATO da viga (ex: "V301")
   • Procura textos com posição EXATA (ex: "N1")
   • Identifica múltiplas ocorrências

2. SELEÇÃO DO CENTRO:
   • Calcula distância entre todos os pares viga+posição
   • Escolhe o par mais próximo
   • Define centro entre os dois textos

3. EXTRAÇÃO REGIONAL:
   • Região focada: 120mm x 90mm (±60mm x, ±45mm y)
   • Renderiza APENAS entidades dentro dessa região
   • Ignora todo o resto do projeto

4. FALLBACKS:
   • Se não encontrar viga+pos juntos, usa um ou outro
   • Se não encontrar textos, busca por layer
   • Se não encontrar layer, busca por bloco
   • Se não encontrar nada, retorna None (não o projeto todo)
