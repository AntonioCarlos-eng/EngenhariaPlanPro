1️⃣  MÉTODO NOVO NO GERADOR:
    
    gerar_e_imprimir_direto(impressora="Argox OS-214 Plus", dpi_x=300, dpi_y=300)
    
    O que faz:
    ├─ Gera etiquetas em arquivo TEMPORÁRIO (não em pasta fixa)
    ├─ Envia CADA etiqueta direto para impressora via Windows Print API
    ├─ Limpa arquivos temporários automaticamente
    ├─ Retorna True/False para sucesso
    └─ Mensagem clara ao usuário

2️⃣  MÉTODO ATUALIZADO NO APP:
    
    _gerar_etiquetas_direto(indices_selecionados=None)
    
    O que faz:
    ├─ Cria GeradorEtiquetasDinamico com dados
    ├─ Filtra apenas as etiquetas selecionadas
    ├─ Chama gerador.gerar_e_imprimir_direto()
    ├─ Recebe confirmação (True = sucesso, False = falha)
    ├─ Mostra mensagem ao usuário
    └─ FIM (sem pasta aberta)
